# AI Loan Underwriting Copilot

An agentic MLOps system for explainable, auditable loan underwriting decisions. A LangChain
agent orchestrates a custom MCP server exposing financial reasoning tools (credit scoring,
debt-to-income calculation, fraud checks, and policy lookup), backed by an MLflow-tracked
XGBoost credit-risk model and a FAISS-based RAG pipeline for grounding decisions in
underwriting policy documents.

Code: GitHub / CI-CD: GitLab

Status: actively in development
